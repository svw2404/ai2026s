作業 05 - Stable Diffusion / ComfyUI

學號：113590032

心得

這次作業讓我第一次比較完整地使用 ComfyUI 來進行 Stable Diffusion 文生圖。剛開始我以為只要下載模型就能直接生成圖片，但實際操作後才發現，Checkpoint、LoRA、Prompt、Negative Prompt、KSampler、VAE Decode 和 Save Image 之間的節點連接都很重要。過程中我也遇到 GPU 記憶體和 NVIDIA driver 的問題，讓我理解到生成解析度、steps、batch size 和 workflow 複雜度都會影響電腦穩定性。最後成功生成芙莉蓮圖片後，我覺得 ComfyUI 的節點式設計雖然一開始比較複雜，但可以清楚看到整個 AI 圖像生成流程，也更容易調整細節。這次作業讓我更了解文生圖模型的實際使用方式，也學到如何處理模型下載、LoRA 使用和工作流調整。
