作業 05

Before working on this assignment, the main image-generation interface I knew was StableDiffusion-WebUI. Through this homework, I learned how to use ComfyUI and understood why it is powerful: it provides a clear node-based workflow, flexible model loading, and better control over checkpoints, prompts, and LoRA connections.

For the simple baseline, I successfully built and ran ComfyUI. For the medium baseline, I generated an image using the selected checkpoint and prompt settings. For the strong baseline, I replaced/extended the model workflow by adding LoRA models and adjusting their strengths to influence the final visual style.

The most difficult part was understanding how the LoRA Loader connects to the model and CLIP nodes, and how the strength values affect the generated result. I also noticed that the prompt quality strongly affects the image: some details around the eyes, hair, and facial expression can still look unnatural if the prompt or parameters are not precise enough. Overall, this assignment helped me understand the practical workflow of AI image generation, especially how checkpoints and LoRA models can be combined in ComfyUI to control style and character appearance.
